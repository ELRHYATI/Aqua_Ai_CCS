# AZURA AQUA - Web Platform (Option B Validée)

## Décision Stratégique
Plateforme web validée par la direction pour:
- Sécurité et conformité M365 natives
- Collaboration centralisée (un modèle, un lien)
- Copilot/IA intégré et performant
- Déploiement contrôlé avec pipelines
- Scalabilité sans impact poste utilisateur

## Prérequis Cybersécurité (OBLIGATOIRE)
- Microsoft Purview DLP activé (licence E5)
- Microsoft Defender for Cloud Apps activé
- Classification des données (CCS - Confidential/Secret/etc.)
- Audit et traçabilité des accès

## Architecture Validée
- Power BI Service (pas Power BI Desktop local)
- Azure App Service pour web app
- Azure AD + Conditional Access
- Power Automate pour workflows
- Azure OpenAI avec gouvernance
- OneDrive comme source (avec DLP)
